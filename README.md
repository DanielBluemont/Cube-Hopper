# Cube-Hopper
 Unity 2D Mobile Game
# Description
* Available on [Google Play](https://play.google.com/store/apps/details?id=com.MoonEyeGames.CubeHopper)
* You can watch gameplay on my [Youtube Channel](https://www.youtube.com/@BluemontStudio/shorts)
* Tutorial on how to play is in game
* Infinite Gameplay,
* Jump on platforms
* Avoid falling, spikes and disappearing platforms
* Collect coins and unclok new skins
* The Gameplay Acompanied by a music written by [Vaavuk](https://github.com/vaavuk)
